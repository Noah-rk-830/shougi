# AI将棋対局アプリ (Powered by 水匠5)

最強クラスの将棋AI「水匠5」を搭載した、シングルプレイ専用の将棋対局Webアプリです。
ブラウザベースの直感的な操作で、AIとの対局を楽しむことができます。

## 特徴

* **最強AI搭載**: 思考エンジンに「水匠5 (YaneuraOu)」を採用。
* **ゲーム内通貨システム**: 勝利してポイントを稼ぎ、ポイントを消費して「待った」を行うゲーム性を導入。
* **インストール不要**: `exe` ファイルを実行するだけで、すぐにブラウザで対局が始まります。
* **自動終了機能**: ブラウザを閉じると、バックグラウンドのアプリも自動的に終了します。

## 起動方法

1.  フォルダ内の **`AIshogi.exe`** をダブルクリックして起動します。
2.  コマンドプロンプト（黒い画面）が立ち上がり、自動的に既定のブラウザで対局画面が開きます。
    * ※開かない場合は、ブラウザのアドレスバーに `http://127.0.0.1:5000` と入力してください。

## 終了方法

* **対局画面（ブラウザ）を閉じるだけです。**
    * ブラウザを閉じると、数秒後にバックグラウンドで動いているアプリケーション（黒い画面）も自動的に終了します。

## 機能説明

### 1. 手合割（ハンデ）設定
対局の手合割（駒落ち）を指定できます。設定を変更すると、即座に盤面がリセットされ新しい対局が始まります。

* **手番について**:
    * **平手**: ユーザーが「先手（▲）」、AIが「後手（△）」となります。
    * **駒落ち**: AIが「先手（駒を落とす側）」、ユーザーが「後手（下側）」となります。

### 2. ポイントシステム
対局の結果やアクションに応じて、ゲーム内ポイントが増減します。

* **獲得**: AIに勝利するとポイントを獲得できます。
* **連勝ボーナス**: 連勝数に応じて、追加のボーナスポイントが付与されます。
* **消費**: 「待った」機能を使用するとポイントを消費します。

### 3. 対局操作
* **待った**:
    * 局面を1手戻します（AIの手と自分の手の計2手分戻ります）。
    * 使用には所持ポイントが必要です。ポイントが不足している場合は使用できません。
* **投了**:
    * 現在の対局における負けを認めます。勝敗数が記録されます。
* **リセット**:
    * 盤面を初期状態に戻します。
    * このボタンによるリセットでは、棋譜の保存や勝敗のカウントは行われません。

## データの保存について

対局の戦績（勝敗数）や所持ポイントは、アプリと同階層にある `save_data` フォルダ内に自動保存されます。
アプリを終了しても、次回起動時に続きから遊ぶことができます。
